Add More Font Styles & Sizes To Your Visual Editor in WordPress

== Description ==

This plugin simply adds 2 buttons to your visual editor:

1. A button for Styles
2. A button for Sizes


== Installation ==

1. Upload `fonts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it. You'll find 2 new buttons added to your visual editor. Go to Add New Posts and check it out!
